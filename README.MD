## Sistema de Biblioteca em Dart

Sistema simples de gerenciamento de livros desenvolvido em Dart, executado diretamente no terminal.

## Funcionalidades

- Cadastrar livros
- Listar livros cadastrados
- Atualizar informações de um livro
- Remover livros com confirmação

## Como executar

No terminal, dentro da pasta do projeto, execute:

dart run

## Como usar o sistema?

Ao executar o programa, será exibido o menu:

=== BIBLIOTECA ===
1 - Cadastrar livro
2 - Listar livros
3 - Atualizar livro
4 - Remover livro
0 - Sair

Digite o número correspondente à ação desejada e pressione ENTER.

## Para cadastrar livros:

Digite 1 e informe os dados solicitados:

Exemplo:
Título: Dom Casmurro
Autor: Machado de Assis
Ano: 1899

O sistema exibirá: "Livro cadastrado com sucesso!"

## Para listar livros: 

Digite 2 para visualizar todos os livros cadastrados:

0 - Dom Casmurro - (Machado de Assis) - 1899

O número no início é o índice do livro.

## Para atualizar informçãoes de livros já existentes: 

Digite 3:

O sistema mostrará a lista de livros
Informe o índice do livro que deseja atualizar
Digite os novos dados

Exemplo:

Digite o índice do livro que deseja atualizar: 0
Novo título: Dom Casmurro (Atualizado)
Novo autor: Machado de Assis
Novo ano: 1900

## Para remover um livro:

Digite 4:

O sistema mostrará os livros
Informe o índice
Confirme a remoção

Tem certeza que deseja remover "Dom Casmurro"? (s/n): s

## Para sair do programa: 

Digite 0 para encerrar o programa.
